# React-MusicPlayer
React音乐播放器

## 效果图
 !["效果图"](./static/images/demo.gif)

## Usage

```bash
# 下载代码
git clone https://github.com/zaizizaizai/MusicPlayer.git
# 进入项目目录，安装依赖
npm install
# 运行预览
npm start
# 开发模式
node server
```
